OpenRouter API Key

Your API key is stored in localStorage in this browser only. It is never sent to any server other than OpenRouter directly from your browser.

How it Works

All model API calls are made directly from your browser through OpenRouter — there is no intermediary server. Your session notes go from your browser to OpenRouter and the routed model provider only. No data is stored by this app externally.

Research sources: The Research Scout uses the Semantic Scholar public API and PubMed E-utilities — both free, no key required. Papers are fetched in real time based on your session themes.
